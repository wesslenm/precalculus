    Section 3.3 Evaluating Trigonometric Functions

    Objectives

    • Evaluate trigonomtric functions using special triangles.
      🔗
      🔗
    • Evaluate trigonomtric functions using the coordinate system and the unit circle.
      🔗
      🔗
    • Evaluate trigonomtric functions using graphs.
      🔗
      🔗
    • Choose an appropriate method to evaluate a trigonomtric function at a given angle.
      🔗
      🔗
    🔗

    Where will this be used in calculus?

    • Whenever trigonometric functions are used, it may also be useful to be able to evaluate them. See Section 3.1 for some notes on where trigonometric functions are used in calculus.
      🔗
      🔗
    • One specific example is when dealing with definite integrals that involve a trigonometric integrand.
      🔗
      🔗
    🔗
    🔗
    In this section we review methods for evaluating all six trigonomtric functions at given angles. Which method to use depends on the type of angle that is given.
    🔗

    Subsection 3.3.1 Evaluating Using Special Triangles

    Being able to evaluate trigonometric functions for common angles will be important in many contexts and examples in calculus. There are several ways to do this. One of the easest is to start with two special triangles.
    🔗

    Aside
    Study tip:
    🔗
    You should take the time to memorize these two special triangles if you haven’t already. They will be useful in finding many trigonometric values. You should practice drawing them on a blank piece of paper until you know them.
    🔗

    Example 3.30. Evaluate using special triangles.

    Use the special triangles to find the value of \(\cos \left(\frac{\pi}{3}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\cos \left(\frac{\pi}{3}\right)= \frac{1}{2}} \end{equation*}
    🔗
    🔗
    Solution.
    We use the special triangle that has the angle \(\frac{\pi}{3} \text{.}\) We also know from the definition of \(\cos \theta \) that \(\cos \theta = \) adjacent / hypothenuse. Reading off the values in the special triangle we see that
    🔗
    \begin{equation*} \cos \left(\frac{\pi}{3}\right) = \frac{adj}{hyp} = \frac{1}{2} \end{equation*}
    The final result is:
    🔗
    \begin{equation*} \boxed{\cos \left(\frac{\pi}{3}\right)= \frac{1}{2}} \end{equation*}
    🔗
    🔗

    Subsection 3.3.2 Evaluating Using the Coordinate System

    If an angle is not in the first quadrant, then it is not enough to use the special triangles. But we can use the special triangles in combination with angles in standard position in a coordinate system.
    🔗
    🔗

    Example 3.34. Evaluate at \(\frac{2\pi}{3}\).

    Find exact values for \(\sin \left(\frac{2\pi}{3}\right), \ \cos \left(\frac{2\pi}{3}\right) \) and \(\tan \left(\frac{2\pi}{3}\right)\) without using a calculator.
    Answer.
    \begin{equation*} \boxed{\sin \left(\frac{2\pi}{3}\right) = \frac{\sqrt{3}}{2}, \cos \left(\frac{2\pi}{3}\right) = - \frac{1}{2}, \tan \left(\frac{2\pi}{3}\right) = - \sqrt{3}} \end{equation*}
    🔗
    🔗
    Solution.
    Start by drawing a coordinate system with a circle centered at the origin (not necessarily the unit circle). Place the angle \(\theta= \frac{2\pi}{3} \) in standard position (see Figure 3.2).
    🔗
    The part of angle \(\frac{2\pi}{3} \) which is in quadrant II, is \(\frac{\pi}{6}\) (because \(\frac{2\pi}{3} - \frac{\pi}{2} = \frac{\pi}{6}\)). Therefore, place the special triangle with \(\frac{\pi}{6}\) (see Figure 3.29) in quadrant II:
    🔗
    Reading from the figure, we see that for the angle \(\theta= \frac{2\pi}{3} \) we can take:
    \begin{equation*} x = -1, y = \sqrt{3}, r = 2. \end{equation*}
    Then, using the definitions in Subsection 3.1.2, we have the following:
    \begin{equation*} \sin \left(\frac{2\pi}{3}\right) = \frac{y}{r} = \frac{\sqrt{3}}{2} \end{equation*}
    \begin{equation*} \cos \left(\frac{2\pi}{3}\right) = \frac{x}{r} = - \frac{1}{2} \end{equation*}
    \begin{equation*} \tan \left(\frac{2\pi}{3}\right) = \frac{x}{r} = - \sqrt{3} \end{equation*}
    🔗
    🔗
    🔗

    Example 3.36. Evaluate at \(-\frac{\pi}{6}\) - HOMEMADE.

    Find exact values for \(\sin \left(-\frac{\pi}{6}\right), \ \cos \left(-\frac{\pi}{6}\right) \) and \(\sec \left(-\frac{\pi}{6}\right)\) without using a calculator.
    Answer.
    \begin{equation*} \boxed{\sin \left(-\frac{\pi}{6}\right) = -\frac{1}{2}, \cos \left(-\frac{\pi}{6}\right) = - \frac{\sqrt{3}}{2}, \sec \left(-\frac{\pi}{6}\right) = - \frac{2}{\sqrt{3}}} \end{equation*}
    🔗
    🔗
    Solution.
    We follow similar steps to the previous example. Start by drawing a coordinate system with a circle, but this time placing the angle \(\theta= -\frac{\pi}{6} \) in standard position (see Figure 3.2). Because it is a negative angle, it is in the clockwise direction.
    🔗
    Place the special triangle with \(\frac{\pi}{6}\) (see Figure 3.29) in quadrant IV:
    🔗
    Reading from the figure, we see that for the angle \(\theta= -\frac{\pi}{6}\) we can take:
    \begin{equation*} x = \sqrt{3}, y = -1, r = 2. \end{equation*}
    Then, using the definitions in Subsection 3.1.2, we have the following:
    \begin{equation*} \sin \left(-\frac{\pi}{6}\right) = \frac{y}{r} = -\frac{1}{2} \end{equation*}
    \begin{equation*} \cos \left(-\frac{\pi}{6}\right) = \frac{x}{r} = - \frac{\sqrt{3}}{2} \end{equation*}
    \begin{equation*} \sec \left(-\frac{\pi}{6}\right) = \frac{r}{x} = - \frac{2}{\sqrt{3}} \end{equation*}
    🔗
    🔗

    Subsection 3.3.3 Evaluating at \(0, \pi/2, \pi\) etc.

    If you know the graphs of \(y=\sin \theta\) and \(y=\cos \theta\text{,}\) then it is easy to find the values of all trigonometric functions for any multiple of \(\frac{\pi}{2}\text{.}\) Trigonometic graphs are discussed in Section 3.2, but the graphs of \(y=\sin \theta\) and \(y=\cos \theta\) are also given here for convenience:
    🔗

    Aside
    Study tip:
    🔗
    You should memorize these graphs, and the graph of \(y=\tan \theta\text{.}\) A lot of information can be seen quickly in a graph, and for most people it’s easier to memorize a graph than a long list of facts.
    🔗

    Example 3.41. Evaluate \(\cos \left(\frac{3\pi}{2}\right)\).

    Evaluate \(\cos \left(\frac{3\pi}{2}\right) \) without using a calculator.
    Answer.
    \begin{equation*} \boxed{\cos \left(\frac{3\pi}{2}\right)= 0} \end{equation*}
    🔗
    🔗
    Solution.
    Option 1: Looking at the graph of \(y=\cos \theta\text{,}\) we see that at \(\frac{3\pi}{2}\text{,}\) the graph crosses the horizontal axis. Therefore the value is \(0\text{.}\)
    \begin{equation*} \boxed{\cos \left(\frac{3\pi}{2}\right)= 0} \end{equation*}
    Option 2: Another option would be to look at angles on the unit circle (in standard position, see Figure 3.2).
    🔗
    From the figure we see that for the angle \(\theta =\frac{3\pi}{2}\text{,}\) we have \(x=0\text{,}\) \(y=-1\) and \(r=1\) (\(r\) is always 1 on the unit circle). Also, \(y=\cos \theta = \frac{x}{r}\) (recall from Subsection 3.1.2). This means \(y=\cos \left(\frac{3\pi}{2}\right) = \frac{0}{1}= 0\text{.}\)
    \begin{equation*} \boxed{\cos \left(\frac{3\pi}{2}\right)= 0} \end{equation*}
    🔗
    🔗
    🔗

    Example 3.42. Evaluate \(\sec \left(4\pi\right)\).

    Evaluate \(\sec \left(4\pi\right)\) without using a calculator.
    Answer.
    \begin{equation*} \boxed{\sec (4\pi)= 1} \end{equation*}
    🔗
    🔗
    Solution.
    We know that \(\sec \theta = \frac{1}{\cos \theta}\) so therefore \(\sec \left(4\pi\right) = \frac{1}{\cos \left(4\pi\right)}\text{.}\) We can evaluate \(\cos (4\pi)\) by looking at the graph of \(\cos \theta\text{,}\) or by using the unit circle. Either way, \(\cos (4\pi)=1\text{.}\) Therefore \(\sec \left(4\pi\right) = \frac{1}{\cos \left(4\pi\right)} = \frac{1}{1} = 1\text{.}\)
    \begin{equation*} \boxed{\sec (4\pi)= 1} \end{equation*}
    🔗
    🔗
    🔗

    Subsection 3.3.4 Summary

    The common angles we have evaluated trigonometric functions at, are summarized here.
    🔗

    Common angles.

    You should be able to evaluate all six trigonometric functions at the following common angles:
    🔗
    • First quadrant: \(\frac{\pi}{6}, \frac{\pi}{4}, \frac{\pi}{3} \) (Subsection 3.3.1)
      🔗
      🔗
    • Corresponding angles in other quadrants, such as \(\frac{5\pi}{6}, \frac{3\pi}{4}, \frac{5\pi}{3}, \) etc. (Subsection 3.3.2)
      🔗
      🔗
    • Multiples of \(\frac{\pi}{2}\text{,}\) such as \(0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi, 3\pi \) etc. (Subsection 3.3.3)
      🔗
      🔗
    • Similar negative angles, such as \(-\frac{\pi}{6}, -\frac{\pi}{2}, -\frac{3\pi}{4} \) etc. (Subsection 3.3.2 and Subsection 3.3.3)
      🔗
      🔗
    🔗
