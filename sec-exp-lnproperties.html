    Where will this be used in calculus?

    Manipulating logarithms into different forms and simplifying using logarithmic properties is useful in many calculus contexts:
    🔗
    • When differentiating using Logarithmic Differentiation
      🔗
      🔗
    • When integrating using Partial Fractions or other integrals involving logarithms
      🔗
      🔗
    • When solving problems involving logarithms and/or exponentials
      🔗
      🔗
    🔗
    🔗
    In Section 2.2 we saw the definition of logarithms and of \(\ln x\text{.}\) In this section we continue working with more properties of logarithms, in particular the Logarithm Laws and properties stemming from the definition as inverse of \(y=e^x\text{.}\)
    🔗

    Subsection 2.3.1 Inverse Properties of Exponentials and Logarithms

    We saw in Section 2.2 that \(y=\ln x\) is the inverse of \(y=e^x\text{.}\) There are several important facts that follow from that. Let’s collect them for easy reference.
    🔗

    Aside
    Study tip:
    🔗
    The properties listed here are important. You should know them well. Spend some time studying them if you don’t know them already.
    🔗

    Properties of Exponentials and Logarithms.

    The following properties follow from the fact that \(y=\ln x\) is the inverse of \(y=e^x\text{:}\)
    🔗
    \begin{equation} y = \ln x \quad \mbox{ if and only if } \quad x = e^y\tag{2.3.1} \end{equation}
    \begin{equation} \ln \left( e^x \right) = x \quad \mbox{ for all values of } x\tag{2.3.2} \end{equation}
    \begin{equation} e^{\ln x} = x \quad \mbox{ for any } x \gt 0\tag{2.3.3} \end{equation}
    \begin{equation} \ln e = 1\tag{2.3.4} \end{equation}
    \begin{equation} \ln 1 = 0 \tag{2.3.5} \end{equation}
    🔗
    🔗

    Example 2.27. Using properties to evaluate.

    Evaluate \(6\ln(e^{3})\) without using a calculator.
    🔗
    Answer.
    \begin{equation*} \boxed{18} \end{equation*}
    🔗
    🔗
    Solution.
    We know from (2.3.2) that \(\ln(e^{x})=x\) so that means \(\ln(e^{3})=3\text{.}\) Therefore
    \begin{equation*} 6\ln(e^{3}) = 6 \cdot 3 = \boxed{18} \end{equation*}
    🔗
    🔗
    🔗

    Subsection 2.3.2 Logarithm Laws

    In Section 1.2, we reviewed the main exponent laws that help us simplify and re-write exponential expressions in simpler ways. Logarithmic expressions also have a similar set of laws that help us re-write logarithmic expressions:
    🔗

    Logarithm Laws.

    For any postitive numbers \(x\) and \(y\) the following is true:
    🔗
    • Product Law:
      \begin{equation} \ln(xy) = \ln x + \ln y \tag{2.3.6} \end{equation}
      🔗
      🔗
    • Quotient Law:
      \begin{equation} \ln\left(\frac{x}{y}\right) = \ln x - \ln y \tag{2.3.7} \end{equation}
      🔗
      🔗
    • Power Law:
      \begin{equation} \ln\left(x^y\right) = y\ln x \tag{2.3.8} \end{equation}
      🔗
      🔗
    🔗
    🔗
    Similar Laws are also true for other logarithms (with other bases) but we will focus on \(\ln x\) (with base \(e\)).
    🔗
    Notation: The notation \(\ln x^y\) means \(\ln\left(x^y\right)\text{.}\) If the intent is to rais the entire logarithm to a power, we would write \(\left(\ln x\right)^y\text{.}\) You should never write \(\ln^a(x)\) (it does not mean anything).
    🔗
    It is important to understand these laws, and it is also important to understand some common errors to avoid when applying these laws:
    🔗

    Common Errors.

    Be careful to not apply `rules’ that are not true. For example, note that:
    🔗
    \begin{equation} \ln (x \pm y) \neq \ln x \pm \ln y\tag{2.3.9} \end{equation}
    \begin{equation} (\ln x)^a \neq a(\ln x)\tag{2.3.10} \end{equation}
    \begin{equation} a\ln x + b\ln y \neq ab\ln\left(xy\right) \tag{2.3.11} \end{equation}
    \begin{equation} \ln\left(xy\right) \neq \ln\left(x\right) \ln\left(y\right) \tag{2.3.12} \end{equation}
    🔗
    🔗

    Example 2.29. True or False.

    Suppose \(a, b,\) and \(c\) are positive numbers. Which of the following are true, and which are false?
    🔗
    Note: For a statement involving \(a, b,\) and/or \(c\) to be true, it must be true for ALL positive numbers \(a, b,\) and \(c\text{.}\)
    🔗
    1. \(\displaystyle \ln (6) + \ln(4.5)=\ln (27)\)
      🔗
      🔗
    2. \(\displaystyle \ln (5x) =\ln (5)\ln (x)\)
      🔗
      🔗
    3. \(\displaystyle \ln \left(\frac{a}{bc}\right)=\ln a - \ln b - \ln c\)
      🔗
      🔗
    🔗
    Answer.
    1. \(\displaystyle \boxed{\mbox{TRUE}}\)
      🔗
      🔗
    2. \(\displaystyle \boxed{\mbox{FALSE}}\)
      🔗
      🔗
    3. \(\displaystyle \boxed{\mbox{TRUE}}\)
      🔗
      🔗
    🔗
    🔗
    Solution.
    1. Using (2.3.6) (uing it `backwards’), we see that \(\ln (6) + \ln(4.5)=\ln (6\cdot 4.6) = \ln (27)\text{.}\) So the statement is \(\boxed{\mbox{TRUE}}\text{.}\)
      🔗
      🔗
    2. \(\ln (5x) =\ln (5)\ln (x)\text{.}\) This is precisely one of the common errors. The statement is \(\boxed{\mbox{FALSE}}\text{.}\)
      🔗
      🔗
    3. First we use (2.3.7):
      \begin{equation*} \ln \left(\frac{a}{bc}\right)=\ln a - \ln (bc) \end{equation*}
      Then we use (2.3.6) on the second term:
      \begin{equation*} \ln \left(\frac{a}{bc}\right)=\ln a - \ln (bc) = \ln a - (\ln b + \ln c) = \ln a - \ln b - \ln c \end{equation*}
      So the statement is \(\boxed{\mbox{TRUE}}\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗

    Example 2.31. Write as one logarithm.

    Write each of the following expressions as a single logarithm, with no coefficient.
    🔗
    1. \(\displaystyle 2 \ln 8 - 5 \ln 2\)
      🔗
      🔗
    2. \(\displaystyle 5\ln (x-2) - \ln (x+5)+3\ln x\)
      🔗
      🔗
    🔗
    Answer.
    1. \(\displaystyle 2 \ln 8 - 5 \ln 2 = \boxed{\ln 2}\)
      🔗
      🔗
    2. \(\displaystyle 5\ln (x-2) - \ln (x+5)+3\ln x = \boxed{\ln \left(\frac{(x-2)^5x^3}{x+5}\right)}\)
      🔗
      🔗
    🔗
    🔗
    Solution.
    Let’s use the logarithm laws several times, and be careful to not use `false laws’:
    1. \(\displaystyle 2 \ln 8 - 5 \ln 2 = \ln 8^2 - \ln 2^5 = \ln \left(\frac{8^2}{2^5} \right) = \ln \left(\frac{64}{32} \right) = \boxed{\ln 2} \)
      🔗
      🔗
    2. \(\displaystyle 5\ln (x-2) - \ln (x+5)+3\ln x = \ln (x-2)^5 + \ln (x+5)^{-1}+\ln x^3 = \ln \left((x-2)^5(x+5)^{-1}x^3\right) = \boxed{\ln \left(\frac{(x-2)^5x^3}{x+5}\right)} \)
      🔗
      🔗
    🔗
    🔗
    🔗
