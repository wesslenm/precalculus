    Section 3.1 Exponential Rules

    Objectives

    • Use exponent rules to simplify and re-work expressions.
    • Simplify roots/radicals without a calculator.
    • Avoid common errors regarding exponents and radicals.

    Where will this be used in calculus?

    Rewriting and simplifying exponentials and radicals is usefully in most computational areas of mathematics, including calculus. Here are two specific calculus examples:
    • When differentiating or integrating a power function (or a combination of power functions), rewrite each term as a single power before differentiating or integrating. For example, rewriting \(f(x) = \frac{1}{x^4}\) as \(f(x) = x^{-4}\) will make it easier to differentiate or integrate.
    • After using rules to differentiate a function (for example the product rule or quotient rule) the result is often a messy complicated looking expression which needs to be simplified.
    Before we dive into the different rules, we need to have a solid understanding of what exponents are and what they mean.
    When we first start learning math, we often start with addition. We then quickly see that repeated addition can be useful. We run into problems like: ``You have 4 dogs and want to give each dog 3 treats. How many treats do you need?’’ We solve these with repeated addition: \(3+3+3+3 = 12\text{,}\) or three treats for each of the four dogs. We then learn that repeated addition happens often, so we develop a new notation, multiplication. For our example problem, we would do \(3 \times 4\) to say we need to add three, four times. Exponents take this one step further. When we need to do repeated multiplication, like if we need to find the volume of a cube, we can shorten the notation by using exponents. To find our volume, we would multiply the side length by itself three times to get length times width times height, but with a cube these lengths are all the same. We can write \(V(x) = (x)(x)(x)=x^3\text{.}\) Here, the exponent tells us how many times to do the multiplication. The concept can later be exanded to include non-integer values of the exponent. But starting by looking at integer exponents can give us an intuitive idea of what exponentials are and how to work with them.

    Subsection 3.1.1 The First Rules

    The first exponent rule we will examine is
    \begin{equation} x^ax^b=x^{a+b}.\tag{3.1.1} \end{equation}
    Here, the first term, \(x^a\) tells us to multiply \(x\) by itself \(a\) times and the second term tells us to multiply it \(b\) times. Together, that says we need to multiply \(x\) a total of \(a+b\) times, giving us \(x^{a+b}\text{.}\) As an example, \(x^2x^3 = x^{2+3} = x^5\text{.}\)

    Checkpoint 3.1.

    Use the properties of exponents to simplify the following
    \({y^{10}}\cdot{y^{5}} =\)
    Note that (3.1.1) is true for all real numbers \(a\) and \(b\) (they do not need to be integers), and all positive values of \(x\text{.}\)
    Next, let’s look at
    \begin{equation*} x^{-a} = \frac{1}{x^a}. \end{equation*}
    This rule builds off of our previous rule. If we have \(x^5x^{-2}\text{,}\) rule (3.1.1) tells us that we really have \(x^{5+(-2)} = x^{5-2} = x^3\text{.}\) We went from having \(x\) multiplied 5 times to having \(x\) multiplied only 3 terms, meaning we have removed two of the multiplications. We removed a multiplication through a division:
    \begin{equation*} \frac{x^5}{x^2} = \frac{(x)(x)(x)(x)(x)}{(x)(x)} = (x)(x)(x) = x^3 \end{equation*}
    A negative exponent tells us we have division rather than multiplication. A more general version of the rule is the following:
    \begin{equation} x^{b-a} = \frac{x^b}{x^a}.\tag{3.1.2} \end{equation}
    We can see this from previous rules:
    \begin{align*} x^{b-a} &= x^{b}x^{-a}\\ &= x^{b} \times \frac{1}{x^a}\\ &= \frac{x^b}{x^a}. \end{align*}
    Again, all of the above is true for all real numbers \(a\) and \(b\text{,}\) and all positive values of \(x\text{.}\)

    Simplify the expression \((11x^5u^2)(7u^3x^4)\text{.}\)

    Subsection 3.1.2 Roots

    Before we continue with more exponent rules, let’s look at the inverse function, or how to ``undo’’ an exponent. These are root functions. A square root ``undoes’’ squaring and a cube root ``undoes’’ cubing. In general, we have
    \begin{equation*} (x^a)^{1/a} = x \end{equation*}
    and
    \begin{equation*} (x^{1/a})^a = x \end{equation*}
    (Both of the above are examples of rule (3.1.5), which we will look more at soon.)
    Often, you will see \(x^{1/a}\) written as \(\sqrt[a]{x}\text{.}\) Mathematicians call \(\sqrt[a]{x}\) the radical form and \(x^{1/a}\) the exponential form. Both have the same meaning. Anytime you see \(\sqrt[a]{x}\text{,}\) you can replace it with \(x^{1/a}\) and vice versa.
    \begin{equation} x^{1/a} = \sqrt[a]{x}\tag{3.1.3} \end{equation}
    Even more generally, we have:
    \begin{equation} x^{b/a} = \sqrt[a]{x^b} = \left( \sqrt[a]{x} \right)^b\tag{3.1.4} \end{equation}
    which is true for all real numbers \(a\) and \(b\text{,}\) and all positive values of \(x\text{.}\)
    Sometimes it is useful to be able to simplify radicals or fractional exponents by hand. Let’s take a look at how we can evaluate these types of terms.

    Example 3.7. Evaluating radicals.

    Evaluate \(8^{2/3}\) without using a calculator.
    Answer.
    \begin{equation*} \boxed{8^{2/3} = 4} \end{equation*}
    Solution.
    \((8^2)^{1/3}\)\((8^{1/3})^2\text{.}\)\((8^2)^{1/3} = (64)^{1/3}\text{,}\)\(64\text{.}\)
    If we use the second version, \((8^{1/3})^2\text{,}\) we would start by finding the cube root of 8. When we factor, we get \(8=2\times2\times2\text{,}\) which show us that \(2=8^{1/3}\text{.}\) This gives us \((8^{1/3})^2 = (2)^2 = 4\text{,}\) so our final answer is
    \begin{equation*} \boxed{8^{2/3} = 4} \end{equation*}

    Subsection 3.1.3 More exponential rules

    The next rule we will discuss is
    \begin{equation} (x^a)^b = x^{ab}.\tag{3.1.5} \end{equation}
    This rule builds directly off of our first rule as well. As a example when \(a\) and \(b\) are integers, \((x^a)^b\) tells us we need to multiply \(x^a\) by itself \(b\) times. Since \(x^a\) multiplies \(x\) by itself \(a\) times, \((x^a)^b\) tells us to multiply \(x\) by itself a total of \(ab\) times. For example,
    \begin{equation*} (x^2)^3 = (x^2)(x^2)(x^2) = x^{2+2+2} = x^{(2)(3)} = x^6. \end{equation*}
    We can also use this rule when there is a product or quotient inside the parentheses, but not if there is an addition or subtraction.
    \begin{equation} (xy)^{a} = x^{a}y^{a}\tag{3.1.6} \end{equation}
    \begin{equation} (xy)^{a} = x^{a}y^{a}\tag{3.1.7} \end{equation}
    However,
    \begin{equation*} (x+y)^a \not= x^a + y^a \end{equation*}

    Example 3.11. Double exponents.

    For example, we can say that
    \begin{equation*} (x^2y^3)^2 = (x^2)^2(y^3)^2 = x^4 y^6, \end{equation*}
    and that
    \begin{equation*} \Bigg (\frac{3x^2}{y^3}\Bigg) ^2 = \frac{(3x^2)^2}{(y^3)^2} = \frac{3^2(x^2)^2}{(y^3)^2} =\frac{9x^4}{y^6}. \end{equation*}
    But it is important to note that \((x^2+y^3)^2 \not= x^4+y^6\text{.}\)

    Example 3.12. Addition inside a bracket.

    Simpliy the expression \((x^2+y^3)^2\text{.}\)

    Answer.
    \begin{equation*} \boxed{(x^2+y^3)^2} = x^4 + 2x^2y^3 + y^6} \end{equation*}

    \begin{align*} (x^2+y^3)^2 &= (x^2+y^3)(x^2+y^3)\\ &= x^2 \times x^2 + x^2y^3 + y^3x^2+ y^3 y^3\\ &= x^4 + 2x^2y^3 + y^6. \end{align*}
    \begin{equation*} \boxed{(x^2+y^3)^2} = x^4 + 2x^2y^3 + y^6} \end{equation*}

    Subsection 3.1.4 Summary

    The rules we have just seen will all be quite handy in calculus (and many other fields of mathematics, of course). In both integral and differential calculus, we will have rules that work well when we have a power function (a function of the form \(f(x)=x^a\)), but won’t work for other forms of functions. By being able to rewrite functions like \(f(x) = \frac{1}{x^4}\text{,}\) in power function form (\(f(x) = x^{-4}\) in this case), other calculations will be simplified.
    Our rules are summarized below.

    Exponential Rules.

    The following are true for all positive numbers \(x\) and \(y\text{,}\) and any real numbers \(a\) and \(b\text{:}\)
    \begin{equation*} x^ax^b=x^{a+b} \end{equation*}
    		\(\hspace{15mm}\) (3.1.1)
    \begin{equation*} x^{a-b} = \frac{x^a}{x^b} \end{equation*}
    		\(\hspace{15mm}\) (3.1.2)
    \begin{equation*} (x^a)^b = x^{ab} \end{equation*}
    		\(\hspace{15mm}\) (3.1.5)
    \begin{equation*} (xy)^{a} = x^{a}y^{a} \end{equation*}
    		\(\hspace{15mm}\) (3.1.6)
    \begin{equation*} \left(\frac{x}{y}\right)^{a} =\frac{x^a}{y^a} \end{equation*}
    		\(\hspace{15mm}\) (3.1.7)

    Notation.

    Suppose \(x\) is any positive number \(x\text{,}\) and \(a\) and \(b\) are integers, with \(a>0\text{.}\) Then:
    • \(\displaystyle x^{1/a} = \sqrt[a]{x}\)
    • \(\displaystyle x^{b/a} = \sqrt[a]{x^b} = \left( \sqrt[a]{x} \right)^b \)
    It is important to know mathematical rules and how to use them, but it is also important to know which `rules’ are not true. Here are a few common errors:

    Common Errors.

    The following expressions are not equal, in general.
    \begin{equation} (x+y)^2 \not= x^2 + y^2\tag{3.1.8} \end{equation}
    \begin{equation} (x+y)^a \not= x^a + y^a\tag{3.1.9} \end{equation}
    \begin{equation} \sqrt{x+y} \not= \sqrt{x} + \sqrt{y}\tag{3.1.10} \end{equation}
    \begin{equation} x^{a}y^{b} \not= (xy)^{ab}\tag{3.1.11} \end{equation}
    \begin{equation} \frac{x^a}{y^b} \not= \left(\frac{x}{y}\right)^{a-b} \tag{3.1.12} \end{equation}

    Subsection 3.1.5 Additional Examples

    Let’s look at a few examples of working with exponent rules. The examples use one or several of the rules we have seen above.
    You are strongly encouraged to try the examples first on your own, before looking at the worked solutions.

    Example 3.17. Simplifying exponents.

    Simplify
    \begin{equation*} \Bigg( \frac{x^2 y^4}{x\sqrt{y}}\Bigg)^2. \end{equation*}
    Answer.
    \begin{equation*} \boxed{\Bigg( \frac{x^2 y^4}{x\sqrt{y}}\Bigg)^2=x^2y^7} \end{equation*}
    Solution.
    \begin{equation*} \Bigg( \frac{x^2 y^4}{x\sqrt{y}}\Bigg)^2 = \Bigg( \frac{x^2 y^4}{xy^{1/2}}\Bigg)^2 \end{equation*}
    \begin{align*} \Bigg( \frac{x^2 y^4}{xy^{1/2}}\Bigg)^2 &= \Big( x^2 y^4 x^{-1} y^{-1/2} \Big)^2\\ &= \Big( x^2 x^{-1} y^4 y^{-1/2} \Big)^2\\ &= \Big( x^{2-1} y^{4-1/2} \Big)^2\\ &= \Big( x^1 y^{8/2-1/2}\Big)^2\\ &= \Big( x y^{7/2})^2\text{.} \end{align*}
    Now that everything inside the parentheses is simplified as much as possible, we will use our third exponent rule to finish simplifying. Our third rule says that \((x^a)^b = x^{ab}\)>. We need to make sure we distribute the exponent that is outside of the parentheses to each term inside of the parentheses. This give us
    \begin{align*} \Big( x y^{7/2}\Big)^2 &= (x)^2 (y^{7/2})^2\\ &= x^2 y^{14/2}\\ &= x^2y^7\text{.} \end{align*}
    So, in the end, we get that
    \begin{equation*} \boxed{\Bigg( \frac{x^2 y^4}{x\sqrt{y}}\Bigg)^2=x^2y^7} \end{equation*}

    Example 3.18. Simplifying exponents involving addition.

    Simplify
    \begin{equation*} \displaystyle \Big( \sqrt{y} + \sqrt{x}\Big)^2. \end{equation*}
    Answer.
    \begin{equation*} \boxed{\Big( \sqrt{y} + \sqrt{x}\Big)^2 = y + 2\sqrt{yx} + x} \end{equation*}
    Solution.
    Note: Be careful to avoid the common error (3.1.8) above.
    We’ll start by focusing on the terms inside the parentheses and rewriting all radicals (roots) as exponents. This gives us
    \begin{equation*} \Big( \sqrt{y} + \sqrt{x} \Big)^2 = \Big( y^{1/2} + x^{1/2} \Big)^2. \end{equation*}
    There is nothing that we can simplify inside the parentheses. We now need to apply the exponent on the outside of the parentheses. Inside the parentheses we have two terms that are added, so we can’t apply an exponent rule here. We will need to rewrite and then expand.
    \begin{align*} \Big(y^{1/2} + x^{1/2} \Big)^2 &= \Big(y^{1/2} + x^{1/2} \Big)\Big(y^{1/2} + x^{1/2} \Big)\\ &= (y^{1/2})^2 + 2 y^{1/2}x^{1/2} + (x^{1/2})^2\\ &= y + 2 y^{1/2}x^{1/2} + x\text{.} \end{align*}
    We don’t have any like terms, so we can’t simplify any further. We could rewrite slightly, but this is a matter of personal preference. We have three other ways we could write this final answer. We could use exponent rules to rewrite the middle term since \(y^{1/2}x^{1/2}=(yx)^{1/2}\text{,}\) giving us \(y+2(yx)^{1/2} + x\text{.}\) We could also use radicals and write either \(y + 2\sqrt{y}\sqrt{x} + x\) or \(y+2\sqrt{yx} + x\text{.}\) All of these four answers are fully simplified, and are equally valid. Probably the most common form is
    \begin{equation*} \boxed{\Big( \sqrt{y} + \sqrt{x}\Big)^2 = y + 2\sqrt{yx} + x} \end{equation*}
