The CAST Rule.

The CAST rule is a memory aid that helps to quickly remember in which quadrants the trigonometric functions are positive or negative. Here is how it works:
🔗
  1. Write out the letters CAST as in the image. Remember to start in Quadrant IV and go counter-clockwise.
    🔗
    🔗
  2. The letters tell you which of the primary trigonometric functions are postive in that quadrant. For example, ``S’’ in Quadrant II indicates that in Quadrant II, \(\sin \theta\) is positive, but \(\cos \theta\) and \(\tan \theta\) are negative.
    🔗
    🔗
  3. The CAST rule reminds you about the signs of \(\sin \theta\text{,}\) \(\cos \theta\) and \(\tan \theta\text{.}\) To find the sign of the other three, use the reciprocal rules. For example, \(\sec \theta = \frac{1}{\cos \theta}\) so \(\sec \theta\) has the same sign as \(\cos \theta\text{,}\) on it’s domain.
    🔗
    🔗
  4. For angles that are not in a quadrant (such as \(\frac{\pi}{2}, \pi\) etc.) each trigonometric function can be positive, negative, 0 or undefined.
