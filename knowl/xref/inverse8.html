Example 3.60. Inverse properties.

Evaluate \(\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) \text{.}\)
🔗
Answer.
\begin{equation*} \boxed{\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) = \frac{2\pi}{3}} \end{equation*}
🔗
🔗
Solution.
The following illuatrates a COMMON ERROR:
🔗
Note that \(\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) \not= \frac{4\pi}{3} \text{.}\) We can not use \(\arccos \left(\cos \left(x\right)\right)=x \) with \(x=\frac{4\pi}{3}\text{,}\) because \(\frac{4\pi}{3}\) is not in the required interval (we need \(0 \leq x \leq \pi\)).
🔗
CORRECT SOLUTION:
🔗
To evaluate \(\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right)\text{,}\) one option is to draw \(\frac{4\pi}{3}\) on the unit cirle (in standard position) and find another angle in the interval \(0 \leq x \leq \pi\) with the same cosine value as \(\cos \left(\frac{4\pi}{3}\right)\text{.}\) Looking at the unit circle, we see that \(\cos \left(\frac{4\pi}{3}\right) = \cos \left(\frac{2\pi}{3}\right)\text{.}\) (See the figure.) This is because by definition, we know that \(\cos \theta = \frac{x}{r}\text{,}\) and both \(x\) and \(r\) are the same for the two angles.
🔗
Therefore:
\begin{equation*} \arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) = \arccos \left(\cos \left(\frac{2\pi}{3}\right)\right) = \frac{2\pi}{3} \end{equation*}
The last step above is true because \(\frac{2\pi}{3}\) is in the interval so now the identity \(\arccos \left(\cos \left(x\right)\right)=x \) can be used.
\begin{equation*} \boxed{\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) = \frac{2\pi}{3}} \end{equation*}
