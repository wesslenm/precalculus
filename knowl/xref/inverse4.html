Example 3.48. Inverse properties.

Evaluate \(\arcsin \left(\sin \left(\pi\right)\right) \text{.}\)
🔗
Answer.
\begin{equation*} \boxed{\arcsin \left(\sin \left(\pi\right)\right) = 0} \end{equation*}
🔗
🔗
Solution.
The following outlines a COMMON ERROR:
🔗
Note that \(\arcsin \left(\sin \left(\pi\right)\right) \not= \pi \text{.}\) We can not use \(\arcsin \left(\sin \left(x\right)\right)=x \) with \(x=\pi\text{,}\) because \(\pi\) is not in the required interval (we need \(-\frac{\pi}{2} \leq x \leq \frac{\pi}{2}\)).
🔗
CORRECT SOLUTION:
🔗
To evaluate \(\arcsin \left(\sin \left(\pi\right)\right)\text{,}\) we first see that \(\sin \left(\pi\right) = 0\text{.}\) So we ned to evaluate \(\arcsin \left(0\right) \text{.}\) Similar to the solutions in Example 3.45 and Example 3.46, this means we need to look for a \(\theta\) in the interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\) such that \(\sin \theta = 0\text{.}\) We easily see that this means \(\theta =0\text{.}\)
\begin{equation*} \boxed{\arcsin \left(\sin \left(\pi\right)\right) = 0} \end{equation*}
