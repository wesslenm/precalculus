Definition of logarithmic functions.

For any positive number \(b\not= 1\text{,}\) the logarithmic function with base \(b\text{,}\) written \(log_b x\) is defined by:
\begin{equation*} y = log_b x \mbox{ if and only if } x = b^y \end{equation*}
In other words, \(y=log_b x\) is the inverse function of \(y = b^x\text{.}\) Note that \(b \gt 0\) and \(b \neq 1\text{.}\)
