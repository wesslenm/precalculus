A function \(y=f(x)\) has an inverse if and only if \(y=f(x)\) is one-to-one. An equavalent way of saying this is that a function \(y=f(x)\) has an inverse if and only if the graph of \(y=f(x)\) passes the horizontal line test (HLT). A function "passes the Horizontal Line tests" if no horizontal line crosses the curve more than once).
