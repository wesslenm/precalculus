Greek letters.

We often (but not always) use Greek letters to represent angles. For example, \(\theta\) (theta), \(\alpha\) (alpha) and \(\beta\) (beta) are commonly used to represent angles.
🔗
For reference, here is a list of the names of some Greek letters commonly used in calculus, in case you are interested.
\begin{align*} \alpha &- alpha\\ \beta &- beta\\ \gamma &- gamma \mbox{ (upper case } \Gamma) \\ \delta &- delta \mbox{ (upper case } \Delta)\\ \epsilon &- epsilon\\ \theta &- theta\\ \lambda &- lambda\\ \mu &- mu\\ \pi &- pi\\ \sigma &- sigma \mbox{ (upper case } \Sigma)\\ \tau &- tau\\ \omega &- omega \mbox{ (upper case } \Omega) \end{align*}
This is not a full list of the Greek alphabet, only the letters most commonly used in calculus and related examples.
