    Section 3.3 Inverse trigonometric functions

    Objectives

    • Evaluate inverse trigonometric functions at common values.
    • Use the correct domain restrictions, and the correct ranges for \(y=\arcsin x, y=\arccos x\) and \(y=\arctan x\text{.}\)
    • Use properties of inverse trigonometric functions to simplify/evaluate expressions.
    • Use correct notation and avoid common mistakes regarding inverse trigonometric functions.

    Where will this be used in calculus?

    Inverse trigonometric functions (and evaluating them) is present in all areas of calculus, such as derivatives, integrals, limits, Taylor series expansion etc. Here are a few examples in calculus and beyond:
    • Derivatives of inverse trigonometric functions appear in physics, particularly in problems involving angles of motion, such as kinematics (certain types of motion), optics (refraction angles) and electrical engineering (arctan functions appear in circuit analysis).
    • Inverse trigonometric functions are used in robotics and computer animation, for example in inverse kinematics used to control robot arms and other manipulators to perform tasks.
    • The derivatives of inverse trigonometric functions are frequently used in implicit differentiation and chain rule applications.
    • Inverse trigonometric functions often appear in integrals involving rational expressions, especially when the denominator is of a certain form.
    In this section we study inverse trigonometric functions. Inverse trigonometric are useful in solving problems involving trigonometric functions, and they are also useful as an additional set of functions (and graphs) which can be used to model and analyze various phenomena.

    Subsection 3.3.1 Arcsine - the inverse sine function

    Let’s start with a reminder regarding inverse function in general:
    A function \(y=f(x)\) has an inverse if and only if \(y=f(x)\) is one-to-one. An equavalent way of saying this is that a function \(y=f(x)\) has an inverse if and only if the graph of \(y=f(x)\) passes the horizontal line test (meaning, no horizontal line crosses the curve more than once).
    Looking at the graph of \(y=\sin x\) in Figure 3.41, we see that it does NOT pass the horizontal line test, so therefore \(y=\sin x\) is not invertible.
    However, this difficulty can be solved by restricting the domain. If we restrict the domain of \(y=\sin x\) to the interval \(\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\text{,}\) then the restricted function is one-to-one. We call the inverse of the restricted sine function, the inverse sine function \(y=\arcsin x\text{.}\)
    Study tip:
    It is very important to note that \(\sin^{-1} x \not= \frac{1}{\sin x}\text{.}\)

    The Inverse Sine Function.

    The inverse sine function, also called arcsine is defined as follows:
    \begin{equation} y = \arcsin x \mbox{ means} \sin y = x\tag{3.3.1} \end{equation}
    where \(-1 \leq x \leq 1\) and \(-\frac{\pi}{2} \leq y \leq \frac{\pi}{2}\text{.}\)
    Some properties of arcsine:
    • \(y = \arcsin x \) is the same as \(y = \sin^{-1} x \text{.}\)
    • The domain of \(y = \arcsin x \) is \([-1,1]\text{.}\)
    • The range of \(y = \arcsin x \) is \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\text{.}\)
    • \(\arcsin (\sin x) = x \) for \(-\frac{\pi}{2} \leq x \leq \frac{\pi}{2}\text{.}\)
    • \(\sin (\arcsin x) = x \) for \(-1 \leq x \leq 1\text{.}\)
    • \(y = \arcsin x \) is an odd function (symmetric about the origin), and \(y = \sin x \) is also an odd function.
    The graph of \(y = \arcsin x \) is:
    Study tip:
    Think of the definition as: if \(y = \arcsin x \text{,}\) then \(y\) is the angle between \(-\frac{\pi}{2}\) and \(\frac{\pi}{2}\) for which \(\sin y = x \text{.}\) Notice that trigonometric functions take angle measures and output real numbers. Inverse trigonometric functions take real numbers and output angle measures.

    Example 3.42. Evaluating arcsine.

    Evaluate \(\arcsin \left(\frac{\sqrt{3}}{2}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arcsin \left(\frac{\sqrt{3}}{2}\right) = \frac{\pi}{3}} \end{equation*}
    Solution.
    Let’s write \(\theta =\arcsin \left(\frac{\sqrt{3}}{2}\right) \text{.}\) From the definition in (3.3.1), this means \(\sin \theta = \frac{\sqrt{3}}{2}\) and \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\)
    Question to consider: Which value of \(\theta\) in the interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\) corresponds to a sine value of \(\frac{\sqrt{3}}{2}\text{?}\) To answer that, we consider the following:
    • The interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\) means that \(\theta\) is in Quadrant I or Quadrant IV.
    • \(\sin x\) is positive in Quadrant I (but not in Quadrant IV).
    • Now place one of the special triangles (see Figure 3.7) in Quadrant I. Choose the triangle and the location that corresponds with \(\sin \theta = \frac{\sqrt{3}}{2} \text{.}\)
    We notice that \(\sin \left(\frac{\pi}{3} \right) = \frac{\sqrt{3}}{2} \text{.}\) Also \(\frac{\pi}{3}\) is in the interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\) Therefore \(\theta = \frac{\pi}{3}\text{.}\)
    \begin{equation*} \boxed{\arcsin \left(\frac{\sqrt{3}}{2}\right) = \frac{\pi}{3}} \end{equation*}

    Example 3.43. Evaluating arcsine.

    Evaluate \(\arcsin \left(-\frac{1}{2}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arcsin \left(-\frac{1}{2}\right) = -\frac{\pi}{6}} \end{equation*}
    Solution.
    Let’s write \(\theta =\arcsin \left(-\frac{1}{2}\right) \text{.}\) From the definition in (3.3.1), this means \(\sin \theta = -\frac{1}{2}\) and \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\)
    Question to consider: Which value of \(\theta\) in the interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\) corresponds to a sine value of \(-\frac{1}{2}\text{?}\) To answer that, we consider the following:
    • The interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\) means that \(\theta\) is in Quadrant I or Quadrant IV.
    • \(\sin x\) is negative in Quadrant VI (but not in Quadrant I). (And we want a negative value since \(\sin \theta = -\frac{1}{2}\text{.}\))
    • Now place one of the special triangles (see Figure 3.7) in Quadrant IV. (see below).
    We see that \(\sin \left(-\frac{\pi}{6} \right) = - \frac{1}{2} \text{.}\) Also \(-\frac{\pi}{6}\) is in the interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\text{.}\) Therefore \(\theta = -\frac{\pi}{6}\text{.}\)
    \begin{equation*} \boxed{\arcsin \left(-\frac{1}{2}\right) = -\frac{\pi}{6}} \end{equation*}

    Example 3.44. Inverse properties.

    Evaluate \(\sin \left(\arcsin \left(\frac{1}{\sqrt{2}}\right)\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\sin \left(\arcsin \left(\frac{1}{\sqrt{2}}\right)\right)= \frac{1}{\sqrt{2}}} \end{equation*}
    Solution.
    Remember that \(\sin \left(\arcsin \left(x\right)\right) =x\) for \(-1\leq x \leq 1\text{.}\)
    Let \(x= \frac{1}{\sqrt{2}}\text{.}\) Since the domain restriction is met (\(\frac{1}{\sqrt{2}}\) is in the interval \([-1,1]\)), this means \(\sin \left(\arcsin \left(\frac{1}{\sqrt{2}}\right)\right) =\frac{1}{\sqrt{2}}\)
    \begin{equation*} \boxed{\sin \left(\arcsin \left(\frac{1}{\sqrt{2}}\right)\right)= \frac{1}{\sqrt{2}}} \end{equation*}

    Example 3.45. Inverse properties.

    Evaluate \(\arcsin \left(\sin \left(\pi\right)\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arcsin \left(\sin \left(\pi\right)\right) = 0} \end{equation*}
    Solution.
    The following is a COMMON ERROR: \(\arcsin \left(\sin \left(\pi\right)\right) \not= \pi \text{.}\) We can not use \(\arcsin \left(\sin \left(x\right)\right)=x \) with \(x=\pi\text{,}\) because \(\pi\) is not in the required interval (we need \(-\frac{\pi}{2} \leq x \leq \frac{\pi}{2}\)).
    CORRECT SOLUTION: To evaluate \(\arcsin \left(\sin \left(\pi\right)\right)\text{,}\) we first see that \(\sin \left(\pi\right) = 0\text{.}\) So we ned to evaluate \(\arcsin \left(0\right) \text{.}\) Similar to the solutions in Example 3.42 and Example 3.43, this means we need to look for a \(\theta\) in the interval \(-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}\) such that \(\sin \theta = 0\text{.}\) We easily see that this means \(\theta =0\text{.}\)
    \begin{equation*} \boxed{\arcsin \left(\sin \left(\pi\right)\right) = 0} \end{equation*}

    Subsection 3.3.2 Arccos - the inverse cosine function

    Looking at the graph of \(y=\cos x\) in Figure 3.50, we see that it also does NOT pass the horizontal line test, so therefore \(y=\cos x\) is not invertible.
    If we restrict the domain of \(y=\cos x\) in exactly the same way as \(y=\sin x\) (namely restrict the domain to \(\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\)), then it is still not oe-to-one. However, we can choose a different interval. The interval we will restrict \(y=\cos x\) to, to make it one-to-one is \(\left[0, \pi\right]\text{.}\) Then the restricted function is one-to-one. We call the inverse of the restricted cosine function, the inverse cosine function \(y=\arccos x\text{.}\)
    Study tip:
    Again \(\cos^{-1} x \not= \frac{1}{\cos x}\text{.}\)

    The Inverse Cosine Function.

    The inverse cosine function, also called arccos, is defined as follows:
    \begin{equation} y = \arccos x \mbox{ means} \cos y = x\tag{3.3.2} \end{equation}
    where \(-1 \leq x \leq 1\) and \(0 \leq y \leq \pi\text{.}\)
    Some properties of arccos:
    • \(y = \arccos x \) is the same as \(y = \cos^{-1} x \text{.}\)
    • The domain of \(y = \arccos x \) is \([-1,1]\text{.}\)
    • The range of \(y = \arccos x \) is \(\left[0,\pi\right]\text{.}\)
    • \(\arccos (\cos x) = x \) for \(0 \leq x \leq \pi\text{.}\)
    • \(\cos (\arccos x) = x \) for \(-1 \leq x \leq 1\text{.}\)
    • \(y = \arccos x \) is neither even nor odd.
    The graph of \(y = \arccos x \) is:
    Study tip:
    Notice that the ranges of \(y = \arccos x \) and \(y = \arcsin x \) are not the same. (They are \(\left[0,\pi\right]\) and \(\left[-\frac{\pi}{2},-\frac{\pi}{2}\right]\) respectively). It is important to memorize these intervals (or remember how to find them).

    Example 3.51. Evaluating arccos.

    Evaluate \(\arccos \left(-\frac{1}{\sqrt{2}}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arccos \left(-\frac{1}{\sqrt{2}}\right) = \frac{3\pi}{4}} \end{equation*}
    Solution.
    Let’s use the notation \(\theta =\arccos \left(-\frac{1}{\sqrt{2}}\right) \text{.}\) From the definition in (3.3.2), this means \(\cos \theta = -\frac{1}{\sqrt{2}}\) and \(0 \leq \theta \leq \pi\text{.}\)
    Question to consider: Which value of \(\theta\) in the interval \(0 \leq \theta \leq \pi\) corresponds to a cosine value of \(-\frac{1}{\sqrt{2}}\text{?}\) To answer that, we consider the following:
    • The interval \(0 \leq \theta \leq \pi\) means that \(\theta\) is in Quadrant I or Quadrant II.
    • \(\cos x\) is negative in Quadrant II (but not in Quadrant I).
    • Using the special triangles (see Figure 3.7), we notice that \(\cos \left(\frac{\pi}{4} \right) = \frac{1}{\sqrt{2}} \text{.}\) To get a negative value, we look in Quadrant II. We see that \(\cos \left(\frac{3\pi}{4} \right) = -\frac{1}{\sqrt{2}} \text{,}\) and also \(\frac{3\pi}{4}\) is in the interval \(0 \leq \theta \leq \pi\text{.}\)
    Therefore \(\theta = \frac{3\pi}{4}\text{.}\)
    \begin{equation*} \boxed{\arccos \left(-\frac{1}{\sqrt{2}}\right) = \frac{3\pi}{4}} \end{equation*}

    Example 3.52. Evaluating arccos.

    Evaluate \(\arccos 0 \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arccos 0 = \frac{\pi}{2}} \end{equation*}
    Solution.
    Start by writing write \(\theta =\arccos 0 \text{.}\) From the definition in (3.3.2), this means \(\cos \theta = 0\) and \(0 \leq \theta \leq \pi\text{.}\)
    Question to consider: Which value of \(\theta\) in the interval \(0 \leq \theta \leq \pi\) corresponds to a cosine value of \(0\text{?}\) The easiest way to answer that might be to look at the graph of \(y=\cos x\) (see Subsection 3.2.1).
    From the graph of we see that \(\cos \left( \frac{\pi}{2}\right) = 0\text{,}\) and \(\frac{\pi}{2}\) is in the interval \(0 \leq \theta \leq \pi\text{.}\)
    Therefore \(\theta = \frac{\pi}{2}\text{.}\)
    \begin{equation*} \boxed{\arccos 0 = \frac{\pi}{2}} \end{equation*}

    Example 3.53. Inverse properties.

    Evaluate \(\cos \left(\arccos \left(-\frac{1}{4}\right)\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\cos \left(\arccos \left(-\frac{1}{4}\right)\right)= -\frac{1}{4}} \end{equation*}
    Solution.
    Remember that \(\cos \left(\arccos \left(x\right)\right) =x\) for \(-1\leq x \leq 1\text{.}\)
    Let \(x= -\frac{1}{4}\text{.}\) Since the domain restriction is met (\(-\frac{1}{4}\) is in the interval \([-1,1]\)), this means \(\cos \left(\arccos \left(-\frac{1}{4}\right)\right) =-\frac{1}{4}\)
    \begin{equation*} \boxed{\cos \left(\arccos \left(-\frac{1}{4}\right)\right)= -\frac{1}{4}} \end{equation*}

    Example 3.54. Inverse properties.

    Evaluate \(\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) = \frac{2\pi}{3}} \end{equation*}
    Solution.
    following is a COMMON ERROR: \(\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) \not= \frac{4\pi}{3} \text{.}\) We can not use \(\arccos \left(\cos \left(x\right)\right)=x \) with \(x=\frac{4\pi}{3}\text{,}\) because \(\frac{4\pi}{3}\) is not in the required interval (we need \(0 \leq x \leq \pi\)).
    CORRECT SOLUTION: To evaluate \(\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right)\text{,}\) one option is to draw \(\frac{4\pi}{3}\) on the unit cirle (in standard position) and find another angle in the interval \(0 \leq x \leq \pi\) with the same cosine value as \(\cos \left(\frac{4\pi}{3}\right)\text{.}\) Looking at the unit circle, we see that \(\cos \left(\frac{4\pi}{3}\right) = \cos \left(\frac{2\pi}{3}\right)\text{.}\) (See the figure.) This is because by definition, we know that \(\cos \theta = \frac{x}{r}\text{,}\) and both \(x\) and \(r\) are the same for the two angles.
    Therefore:
    \begin{equation*} \arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) = \arccos \left(\cos \left(\frac{2\pi}{3}\right)\right) = \frac{2\pi}{3} \end{equation*}
    The last step above is true because \(\frac{2\pi}{3}\) is in the interval so now the identity \(\arccos \left(\cos \left(x\right)\right)=x \) can be used.
    \begin{equation*} \boxed{\arccos \left(\cos \left(\frac{4\pi}{3}\right)\right) = \frac{2\pi}{3}} \end{equation*}

    Subsection 3.3.3 Arctan - the inverse tangent function

    Just like \(y=\sin x\) and \(y=\cos x\) are not one-to-one, we see that the graph of \(y=\tan x\) also does NOT pass the horizontal line test, so therefore \(y=\tan x\) is also not invertible on its full domain.
    This time, if we restrict the domain to \(\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\) then \(y=\tan x\) is one-to-one on that domain. We call the inverse of the restricted tangent function, the inverse tangent function \(y=\arctan x\text{.}\)
    Study tip:
    Again \(\tan^{-1} x \not= \frac{1}{\tan x}\text{.}\)

    The Inverse Tangent Function.

    The inverse tangent function, also called arctan, is defined as follows:
    \begin{equation} y = \arctan x \mbox{ means} \tan y = x\tag{3.3.3} \end{equation}
    where \(-\infty \lt x \lt \infty\) and \(-\frac{\pi}{2} \lt y \lt \frac{\pi}{2}\text{.}\)
    Some properties of arctan:
    • \(y = \arctan x \) is the same as \(y = \tan^{-1} x \text{.}\)
    • The domain of \(y = \arctan x \) is \((-\infty,\infty)\text{.}\)
    • The range of \(y = \arctan x \) is \(\left(-\frac{\pi}{2},\frac{\pi}{2}\right)\text{.}\)
    • \(\arctan (\tan x) = x \) for \(\frac{\pi}{2} \leq x \leq \frac{\pi}{2}\text{.}\)
    • \(\tan (\arctan x) = x \) for all real \(x\text{.}\)
    • \(y = \arctan x \) is an odd function.

    Study tip:
    It is important to memorize the ranges of \(\arctan x, \arcsin x\) and \(\arccos x\) (or remember how to find them). The graph of \(y=\arctan x\) also comes up often, and you should memorize it, including the values of the asymptotes (\(y=\frac{\pi}{2}\) and \(y=-\frac{\pi}{2}\)).

    Example 3.60. Evaluating arctan.

    Evaluate \(\arctan \left(\sqrt{3}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arctan \left(\sqrt{3}\right) = \frac{\pi}{3}} \end{equation*}
    Solution.
    Let \(\theta =\arctan \left(\sqrt{3}\right) \text{.}\) From the definition in (3.3.3), this means \(\tan \theta = \sqrt{3}\) and \(\theta \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\text{.}\)
    Question to consider: Which value of \(\theta\) in the interval \(\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\) corresponds to a tangent value of \(\sqrt{3}\text{?}\) Consider the following:
    • The interval \(\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\) means that \(\theta\) is in Quadrant I or Quadrant IV.
    • \(\tan x\) is positive in Quadrant I (but not in Quadrant IV).
    • Therefore, place a special triangle (see (2.1.1)) in Quadrant IV as in Subsection 3.1.3.
    We see that \(\tan \left(\frac{\pi}{3} \right) = \sqrt{3} \text{.}\) Also, \(\frac{\pi}{3}\) is in the interval \(\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\text{.}\) Therefore \(\theta = \frac{\pi}{3}\text{.}\)
    \begin{equation*} \boxed{\arctan \left(\sqrt{3}\right) = \frac{\pi}{3}} \end{equation*}

    Subsection 3.3.4 The remaining three inverse trigonometric functions

    Similarly to \(y=\sin x, y=\cos x\) and \(y=\tan x\text{,}\) we can see by looking at their graphs that none of \(y=\csc x, y=\sec x\) and \(y=\cot x\) are invertible either. Again we need to restrict their domains to create invertible functions. The definitions are similar to what we have seen in the previous sectiobs. Let’s summarize the results here.

    Summary of all 6 inverse trigonometric functions.

    • Inverse function: \(y= \arcsin x = \sin^{-1} x\)
      Domain: \([-1,1]\)
      Range: \(\left[-\frac{\pi}{2},\frac{\pi}{2} \right] \)
    • Inverse function: \(y= \arccos x = \cos^{-1} x\)
      Domain: \([-1,1]\)
      Range: \(\left[0,\pi \right] \)
    • Inverse function: \(y= \arctan x = \tan^{-1} x\)
      Domain: \((-\infty,\infty)\)
      Range: \(\left(-\frac{\pi}{2},\frac{\pi}{2} \right) \)
    • Inverse function: \(y= \mbox{arccot } x = \cot^{-1} x\)
      Domain: \((-\infty,\infty)\)
      Range: \(\left(0,\pi \right) \)
    • Inverse function: \(y= \mbox{arcsec } x = \sec^{-1} x\)
      Domain: \((-\infty,-1] \cup [1,\infty)\)
      Range: \(\left[0,\frac{\pi}{2}\right) \cup \left(\frac{\pi}{2}, \pi \right] \)
    • Inverse function: \(y= \mbox{arccsc } x = \csc^{-1} x\)
      Domain: \((-\infty,-1] \cup [1,\infty)\)
      Range: \(\left[-\frac{\pi}{2},0\right) \cup \left(0, \frac{\pi}{2} \right] \)

    Example 3.62. Evaluating arccot.

    Evaluate arccot \(\left(\sqrt{3}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\mbox{arccot} \left(\sqrt{3}\right) = \frac{\pi}{6}} \end{equation*}
    Solution.
    Let \(\theta =\mbox{arccot} \left(\sqrt{3}\right) \text{.}\) This means \(\cot \theta = \sqrt{3}\) and \(\theta \in \left(0, \pi\right)\text{.}\)
    Which value of \(\theta\) in the interval \(\theta \in \left(0, \pi\right)\) corresponds to a cotangent value of \(\sqrt{3}\text{?}\)
    • The interval \(\theta \in \left(0, \pi\right)\) means that \(\theta\) is in Quadrant I or Quadrant II.
    • \(cot x\) is positive in Quadrant I (but not in Quadrant II).
    • Using the special triangles in Subsection 3.1.3, we notice that \(\cot \left(\frac{\pi}{6} \right) = \sqrt{3} \) and \(\frac{\pi}{6}\) is in the interval \(\theta \in \left(0, \pi\right)\text{.}\)
    Therefore \(\theta = \frac{\pi}{6}\text{.}\)
    \begin{equation*} \boxed{\mbox{arccot} \left(\sqrt{3}\right) = \frac{\pi}{6}} \end{equation*}

    Common errors.

    Be careful to not make the following errors. It is true that \(\cot x = \dfrac{1}{\tan x}\) but note that:
    \begin{equation*} \mbox{arccot } x \not= \frac{1}{\arctan x} \end{equation*}
    \begin{equation*} \mbox{arcsec } x \not= \frac{1}{\arccos x} \end{equation*}
    \begin{equation*} \mbox{arccsc } x \not= \frac{1}{\arcsin x} \end{equation*}
    The box below shows the corresponding correct identities (3.3.4), (3.3.5) and (3.3.6).

    Inverse identities.

    The following identities hold.
    \begin{equation} \mbox{arcsec } x = \arccos \left( \frac{1}{x}\right) \mbox{ for } x \leq -1 \mbox{ or } x \geq 1\tag{3.3.4} \end{equation}
    \begin{equation} \mbox{arccsc } x = \arcsin \left( \frac{1}{x}\right) \mbox{ for } x \leq -1 \mbox{ or } x \geq 1\tag{3.3.5} \end{equation}
    \begin{equation} \mbox{arccot } x = \left\{ \begin{array} {cc} \arctan \left( \frac{1}{x}\right) \mbox{ for } x >0 \\ \pi + \arctan \left( \frac{1}{x}\right) \mbox{ for } 0>x \\ \end{array} \right.\tag{3.3.6} \end{equation}
    Study tip:
    Most mathematicians do not memorize these identities, but may look them up if needed.

    Example 3.65. Evaluating arccos.

    Evaluate \(\arccos \left(-\frac{\sqrt{3}}{2}\right) \text{.}\)
    Answer.
    \begin{equation*} \boxed{\arccos \left(-\frac{\sqrt{3}}{2}\right) = \frac{5\pi}{6}} \end{equation*}
    Solution.
    Let \(\theta =\arccos \left(-\frac{\sqrt{3}}{2}\right)\text{.}\) The range of \(\arccos \left(-\frac{\sqrt{3}}{2}\right) \) is \(\left[0, \frac{\pi}{2}\right]\) (you should memorize that!). Therefore we have need to find a \(\theta\) such that \(\cos \theta = -\frac{\sqrt{3}}{2}\) and \(0 \leq \theta \leq \frac{\pi}{2}\text{.}\)
    Consider the following:
    • The interval \(\left[0, \frac{\pi}{2}\right]\) means that \(\theta\) is in Quadrant I or Quadrant II.
    • \(cos x\) is negative in Quadrant II (but not in Quadrant I), so we are looking for a value on Quadrant II.
    • Using the special triangles in Subsection 3.1.3, we notice that \(\cos \left(\frac{5\pi}{6}\right) = -\frac{\sqrt{3}}{2}\text{.}\) Also, \(\frac{5\pi}{6}\) is in the interval \(\left[0, \frac{\pi}{2}\right]\text{.}\)
    Therefore \(\theta = \frac{5\pi}{6}\text{.}\)
    \begin{equation*} \boxed{\arccos \left(-\frac{\sqrt{3}}{2}\right) = \frac{5\pi}{6}} \end{equation*}

    Example 3.66. Correct or incorrect?

    Have a look at the following reasoning. Is this correct or incorrect? If it is incorrect, find all mistakes.
    Question: Evaluate \(\arcsin \left( \sin \left( \frac{5\pi}{7}\right) \right) \)
    `Solution’: Recall that \(\arcsin \left( \sin x \right) = x\) for \(0 \leq x \leq \pi\text{.}\) Since \(0 \leq \frac{5\pi}{7} \leq \pi\) this means \(\arcsin \left( \sin \left( \frac{5\pi}{7}\right) \right) = \frac{5\pi}{7}\text{.}\)
    Answer.
    It is incorrect.
    Solution.
    The `solution’ is incorrect. The range of \(\arcsin x\) is \(\left[ - \frac{\pi}{2}, \frac{\pi}{2}\right] \) so the identity \(\arcsin \left( \sin x \right) = x\) holds for \(- \frac{\pi}{2} \leq x \leq \frac{\pi}{2}\) (not for \(0 \leq \frac{5\pi}{7} \leq \pi\) ).

    Example 3.67. Correct or incorrect?

    Have a look at the following reasoning. Is this correct or incorrect? If it is incorrect, find all mistakes.
    Question: Evaluate \(\mbox{arccot} \left(-\sqrt{3} \right) \)
    `Solution’:
    \begin{equation*} \mbox{arccot} \left(-\sqrt{3} \right) = \frac{1}{\arctan \left(-\sqrt{3} \right)} \end{equation*}
    Now, let \(\theta = \arctan \left(-\sqrt{3} \right) \) so that \(\tan \theta = -\sqrt{3} \) and \(\theta \in \left(- \frac{\pi}{2}, \frac{\pi}{2}\right)\text{.}\) By using a special triangle in Quadrant IV, we see that \(\tan \left(-\frac{\pi}{3}\right) = -\sqrt{3} \) and \(-\frac{\pi}{3} \in \left(- \frac{\pi}{2}, \frac{\pi}{2}\right)\text{.}\)
    Therefore \(\mbox{arccot} \left(-\sqrt{3} \right) = -\frac{\pi}{3}\) so then
    \begin{equation*} \mbox{arccot} \left(-\sqrt{3} \right) = \frac{1}{\arctan \left(-\sqrt{3} \right)} = \frac{1}{-\frac{\pi}{3}} = -\frac{3}{\pi} \end{equation*}
    Answer.
    It is incorrect.
    Solution.
    The `solution’ is incorrect. The first line is incorrect, because
    \begin{equation*} \mbox{arccot} \left(-\sqrt{3} \right) \not= \frac{1}{\arctan \left(-\sqrt{3} \right)} \end{equation*}
    Instead, solve the problrm similarly to Example 3.62. The correct final answer is
    \begin{equation*} \mbox{arccot} \left(-\sqrt{3} \right) = \frac{5\pi}{6}} \end{equation*}
