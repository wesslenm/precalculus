    Section 3.4 Trigonometric Identities

    Objectives

    • Describe the difference between a trigonometric identity and a trigonometric equation.
      🔗
      🔗
    • Recall the most common trigonometric identities, and be familiar with some additional commonly used trigonometric identities.
      🔗
      🔗
    • Use trigonometric identities to simplify and re-write trigonometric expressions.
      🔗
      🔗
    • Use graphing tools effectively.
      🔗
      🔗
    🔗

    Where will this be used in calculus?

    Trigonometric identities appear throughout calculus because they allow expressions involving trigonometric functions to be rewritten into simpler or more useful forms. They often make otherwise difficult problems more manageable. Some examples include:
    🔗
    • Simplifying derivatives
      🔗
    • Rewriting integrands so that integration techniques can be applied
      🔗
    • Converting products or powers of trigonometric functions into sums that are easier to integrate
      🔗
    • Transforming expressions into recognizable patterns, to prepare for integration by substitution
      🔗
    • Simplifying expressions in limit problems
      🔗
    • Reducing complexity during multi-step calculus manipulations
      🔗
    🔗
    🔗
    In this section we will discuss what trigonometic identities are, and practice using them. There are numerous identities, and we will not discuss all of them here, but we will focus on the most common ones.
    🔗

    Subsection 3.4.1 What is a Trigonometric Identity?

    A trigonometric identity is an equation involving trigonometric functions that is true for all values where the functions are defined. These identities are used to simplify expressions or rewrite them in equivalent forms.
    🔗
    Unlike conditional equations, which are only true for specific values, identities hold for all values within the domain.
    🔗
    One of the most fundamental examples is the Pythagorean identity:
    🔗
    \begin{equation*} \sin^2(x) + \cos^2(x) = 1 \end{equation*}
    We can check this identity by substituting values for \(x\text{.}\) For example, let \(x = 0\text{:}\)
    🔗
    \begin{equation*} \sin(0) = 0, \quad \cos(0) = 1, \quad \mbox{ and } 0^2 + 1^2 = 1 \end{equation*}
    Next, for example let \(x = \frac{\pi}{4}\text{:}\)
    🔗
    \begin{equation*} \sin\left(\frac{\pi}{4}\right) = \frac{1}{\sqrt{2}}, \quad \cos\left(\frac{\pi}{4}\right) = \frac{1}{\sqrt{2}} \end{equation*}
    \begin{equation*} \left(\frac{\sqrt{2}}{2}\right)^2 + \left(\frac{\sqrt{2}}{2}\right)^2 = \frac{1}{2} + \frac{1}{2} = 1 \end{equation*}
    Of course we can not check all values of \(x\) one by one (there are infinitely many). But using the unit circle and definition of trigonometric functions, it is possible to prove that it holds for all values of \(x\text{.}\) Since the equation holds for all valid values of \(x\text{,}\) it is called an identity.
    🔗
    🔗

    Subsection 3.4.2 Fundamental Trigonometric Identities

    The following families of identities form the core toolkit used throughout trigonometry and calculus.
    🔗

    Aside
    Study tip:
    🔗
    The notation \(\sin^{2} x\) means \(\sin x\) raised to the power of \(2\text{.}\) In other words \(\sin^{2} x = \left(\sin x\right)^2\text{.}\)
    🔗

    Fundamental Trigonometric Identities.

    Pythagorean Identities:
    \begin{equation} \sin^2 x + \cos^2 x = 1\tag{3.4.1} \end{equation}
    \begin{equation} \tan^2x +1 = \sec^2 x\tag{3.4.2} \end{equation}
    \begin{equation} 1 + \cot^2 x = \csc^2 x\tag{3.4.3} \end{equation}
    🔗
    Reciprocal Identities:
    \begin{equation} \sec x = \frac{1}{\cos x}\tag{3.4.4} \end{equation}
    \begin{equation} \csc x = \frac{1}{\sin x}\tag{3.4.5} \end{equation}
    \begin{equation} \cot x = \frac{1}{\tan x}\tag{3.4.6} \end{equation}
    🔗
    Quotient Identities:
    \begin{equation} \tan x = \frac{\sin x}{\cos x}\tag{3.4.7} \end{equation}
    \begin{equation} \cot x = \frac{\cos x}{\sin x}\tag{3.4.8} \end{equation}
    🔗
    Even-Odd Properties:
    \begin{equation} \sin(-x) = -\sin x \tag{3.4.9} \end{equation}
    \begin{equation} \cos(-x) = \cos x\tag{3.4.10} \end{equation}
    \begin{equation} \tan(-x) = -\tan x\tag{3.4.11} \end{equation}
    🔗
    🔗

    Aside
    Study tip:
    🔗
    Read these tips that help you know many trig identities by just memorizing a few of them.
    🔗
    Some of the identities above are best to memorize, such as (3.4.1), (3.4.4), (3.4.5), (3.4.6) and (3.4.7). You likely know some of those already. However, many of the remaining ones you can get very easily from information you already know. Here are some examples:
    🔗
    • (3.4.2): Start with (3.4.1) and divide both sides by \(\cos^2 x\text{.}\)
      🔗
    • (3.4.3): Start with (3.4.1) and divide both sides by \(\sin^2 x\text{.}\)
      🔗
    • (3.4.8): Start with (3.4.7) and take the reciprocal of both sides.
      🔗
    • The Even-Odd Properties: These come from the fact that \(y=\sin x\) and \(y=\tan x\) are odd while \(y=\cos x\) is even (see Subsection 3.2.3).
      🔗
    🔗

    Example 3.46. Evaluate.

    Suppose that for some angle \(\theta\) we know that \(\sin \theta = -\frac{5}{13}\) and \(\cos \theta = \frac{12}{13}\text{.}\) Find the following values.
    1. \(\displaystyle \sec \theta\)
      🔗
      🔗
    2. \(\displaystyle \cot \theta\)
      🔗
      🔗
    3. \(\displaystyle \sin^2 \theta\)
      🔗
      🔗
    🔗
    Answer.
    1. \(\displaystyle \sec \theta = \boxed{\frac{13}{12}}\)
      🔗
      🔗
    2. \(\displaystyle \cot \theta = \boxed{\frac{12}{5}}\)
      🔗
      🔗
    3. \(\displaystyle \sin^2 \theta = \boxed{\frac{25}{169}}\)
      🔗
      🔗
    🔗
    🔗
    Solution.
    1. From (3.4.4) we know that \(\sec \theta = \frac{1}{\cos \theta}\text{.}\) So with the given information we see that:
      \begin{equation*} \sec \theta = \frac{1}{\cos \theta} = \frac{1}{\frac{12}{13}} = \boxed{\frac{13}{12}} \end{equation*}
      🔗
      🔗
    2. From (3.4.8) we know that \(\cot \theta = \frac{\cos \theta}{\sin \theta}\text{.}\) So with the given information we see that:
      \begin{equation*} \cot \theta = \frac{\cos \theta}{\sin \theta} = \frac{\frac{12}{13}}{-\frac{5}{13}} = - \frac{12}{13} \times \frac{13}{5} =\boxed{\frac{12}{5}} \end{equation*}
      🔗
      🔗
    3. Here we don’t need an identity.
      \begin{equation*} \sin^2 \theta = \left(\sin x\right)^2 = \left(-\frac{5}{13}\right)^2 = \frac{5^2}{13^2} = \boxed{\frac{25}{169}} \end{equation*}
      🔗
      🔗
    🔗
    🔗
    🔗

    Example 3.48. Evaluate at an unknown angle.

    Suppose \(\theta\) is an angle with terminal side in Quadrant IV, and suppose \(\cos \theta = \frac{2}{5}\text{.}\) What is the value of \(\sin \theta\text{?}\)
    🔗
    Answer.
    \begin{equation*} \sin \theta = \boxed{-\frac{\sqrt{21}}{5}} \end{equation*}
    🔗
    🔗
    Solution.
    We can use the Pythagorean Identity (3.4.1):
    \begin{equation*} \sin^2 \theta + \cos^2 \theta = 1 \end{equation*}
    Then insert the known value for \(\cos \theta = \frac{2}{5}\text{:}\)
    \begin{equation*} \sin^2 \theta + \left(\frac{2}{5}\right)^2 = 1 \end{equation*}
    Simplify
    \begin{equation*} \sin^2 \theta = 1 - \left(\frac{2}{5}\right)^2 \end{equation*}
    \begin{equation*} \sin^2 \theta = \frac{25}{25} - \frac{4}{25} \end{equation*}
    \begin{equation*} \sin^2 \theta = \frac{21}{25} \end{equation*}
    Then take square roots (recall that \(\sin^2 x = \left(\sin x\right)^2\)):
    \begin{equation*} \sin \theta = \pm \sqrt{\frac{21}{25}} \end{equation*}
    \begin{equation*} \sin \theta = \pm \frac{\sqrt{21}}{\sqrt{25}} \end{equation*}
    \begin{equation*} \sin \theta = \pm \frac{\sqrt{21}}{5} \end{equation*}
    To determine whether we should use the positive or the negative value, we should consider signs of trigonometric functions in different quadrants. For example, we can use the CAST rule (Assemblage ) to see that \(\sin \theta\) is negative in Quadrant IV. Recall that the question specified that \(\theta\) is in Quadrant IV. Therefore
    \begin{equation*} \sin \theta = \boxed{-\frac{\sqrt{21}}{5}} \end{equation*}
    🔗
    🔗
    🔗

    Example 3.50. Simplifying.

    Simplify the expressions as much as possible and write your final answers in terms of \(\sin \theta\) and \(\cos \theta\) only.
    1. \(\displaystyle \frac{\csc^2\theta - \cot^2\theta}{\sec^2\theta}\)
      🔗
      🔗
    2. \(\displaystyle \frac{1 - \cos^2\theta}{\sin\theta} + \frac{\sin\theta\tan\theta}{\sec\theta}\)
      🔗
      🔗
    🔗
    Answer.
    1. \(\displaystyle \frac{\csc^2\theta - \cot^2\theta}{\sec^2\theta} = \boxed{\cos^2\theta}\)
      🔗
      🔗
    2. \(\frac{1 - \cos^2\theta}{\sin\theta} + \frac{\sin\theta\tan\theta}{\sec\theta} = \boxed{\sin\theta + \sin^2\theta}\) or \(\boxed{\sin \theta(1+ \sin\theta)}\text{.}\)
      🔗
      🔗
    🔗
    🔗
    Solution.
    1. First use (3.4.3) on the numerator:
      \begin{equation*} \frac{\csc^2\theta - \cot^2\theta}{\sec^2\theta} = \frac{1}{\sec^2\theta} \end{equation*}
      then use (3.4.4) and simplify:
      \begin{equation*} \frac{1}{\sec^2\theta} = \frac{1}{\frac{1}{\cos^2\theta}} = \boxed{\cos^2\theta} \end{equation*}
      🔗
      🔗
    2. First use (3.4.1) on the first numerator. Also use (3.4.7) and (3.4.4) to write everything in terms of \(\sin \theta\) and \(\cos \theta\text{:}\)
      \begin{equation*} \frac{1 - \cos^2\theta}{\sin\theta} + \frac{\sin\theta\tan\theta}{\sec\theta} = \frac{\sin^2\theta}{\sin\theta} + \frac{\sin\theta\frac{\sin\theta}{\cos\theta}}{\frac{1}{\cos\theta}} \end{equation*}
      Cancel common factor \(\sin \theta\) in the first term. In the second term, multiply by the reciprical:
      \begin{equation*} = \sin\theta + \sin\theta\frac{\sin\theta}{\cos\theta} \times\cos\theta \end{equation*}
      Cancel \(\cos \theta\)
      \begin{equation*} = \sin\theta + \sin\theta \times\sin\theta \end{equation*}
      \begin{equation*} = \sin\theta + \sin^2\theta \end{equation*}
      We could factor \(\sin \theta\) to get \(\sin \theta(1+ \sin\theta)\) but it is not any simpler than \(\sin\theta + \sin^2\theta\text{.}\) Both of those are good final answers.
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection 3.4.3 Additional Trigonometric Identities

    Here we list some additional identities. These are far from all identities. There are many other ones as well, and some of the ones missing here are commonly used. However, we have included those that are most commonly used in calculus.
    🔗

    Additional Trigonometric Identities.

    Sum Identities:
    \begin{equation} \sin(x+y) = \sin x \cos y + \cos x \sin y\tag{3.4.12} \end{equation}
    \begin{equation} \cos(x+y) = \cos x \cos y - \sin x \sin y\tag{3.4.13} \end{equation}
    \begin{equation} \tan(x+y) = \frac{\tan x + \tan y}{1 - \tan x \tan y}\tag{3.4.14} \end{equation}
    🔗
    Difference Identities:
    \begin{equation} \sin(x-y) = \sin x \cos y - \cos x \sin y\tag{3.4.15} \end{equation}
    \begin{equation} \cos(x-y) = \cos x \cos y - \sin x \sin y\tag{3.4.16} \end{equation}
    \begin{equation} \tan(x-y) = \frac{\tan x - \tan y}{1 + \tan x \tan y}\tag{3.4.17} \end{equation}
    🔗
    Double-Angle Identities:
    \begin{equation} \sin 2x = 2\sin x \cos y \tag{3.4.18} \end{equation}
    \begin{equation} \cos 2x = \cos x \cos y - \sin x \sin y\tag{3.4.19} \end{equation}
    \begin{equation} \tan 2x = \frac{\tan x + \tan y}{1 - \tan x \tan y}\tag{3.4.20} \end{equation}
    🔗
    Identities for reducing powers:
    \begin{equation} \sin^2x = \frac{1- \cos (2x)}{2} \tag{3.4.21} \end{equation}
    \begin{equation} \cos^2x = \frac{1+ \cos (2x)}{2}\tag{3.4.22} \end{equation}
    \begin{equation} \tan^2x = \frac{1- \cos (2x)}{1+ \cos (2x)}\tag{3.4.23} \end{equation}
    🔗
    Product-Sum Identities:
    \begin{equation} \sin x \cos y = \frac{1}{2} \left( \sin(x+y) + \sin(x-y)\right) \tag{3.4.24} \end{equation}
    \begin{equation} \cos x \sin y = \frac{1}{2} \left( \sin(x+y) - \sin(x-y)\right) \tag{3.4.25} \end{equation}
    \begin{equation} \sin x \sin y = \frac{1}{2} \left( \cos(x-y) - \cos(x+y)\right) \tag{3.4.26} \end{equation}
    \begin{equation} \cos x \cos y = \frac{1}{2} \left( \cos(x+y) + \cos(x-y)\right) \tag{3.4.27} \end{equation}
    🔗
    🔗

    Example 3.52. Simplify - HOMEMADE.

    Simplify the expressions as much as possible.
    \begin{equation*} \cos\left(\frac{\pi}{3}x\right)\cos\left(-\frac{\pi}{2}x\right)-\sin\left(\frac{\pi}{3}x\right)\sin\left(-\frac{\pi}{2}x\right) \end{equation*}
    🔗
    Answer.
    \begin{equation*} \boxed{\cos\left(\frac{5\pi}{6}x\right)} \end{equation*}
    🔗
    🔗
    Solution.
    Use the identity (3.4.16):
    \begin{equation*} \cos\left(\frac{\pi}{3}x\right)\cos\left(-\frac{\pi}{2}x\right)-\sin\left(\frac{\pi}{3}x\right)\sin\left(-\frac{\pi}{2}x\right) = \cos\left(\frac{\pi}{3}x+\frac{\pi}{2}x\right) \end{equation*}
    Then simplify by adding the fractions (write on common denominator Subsection 1.1.3) and simplify:
    \begin{equation*} = \cos\left(\frac{2\pi x}{6}+\frac{3 \pi x}{6}\right) = \cos\left(\frac{5\pi x}{6}\right) =\boxed{\cos\left(\frac{5\pi x}{6}\right)} \end{equation*}
    🔗

    Example 3.54. Simplify - HOMEMADE.

    Re-write the below expression so that the largest power is 1, and so that no trigonometric function is multipled by a trigonometric function.
    \begin{equation*} \cos^2 x \sin^2 x \end{equation*}
    🔗
    Answer.
    \begin{equation*} \boxed{\frac{1 - \cos(4x)}{8}} \end{equation*}
    🔗
    🔗
    Solution.
    Use the identities (3.4.21) and (3.4.22):
    \begin{equation*} \cos^2 x \sin^4 x = \cos^2 x \sin^2 x = \left(\frac{1 + \cos(2x)}{2}\right)\left(\frac{1 - \cos(2x)}{2}\right) \end{equation*}
    Expand the brackets to get a difference of squares:
    \begin{equation*} \frac{1 - \cos^2(2x)}{4} \end{equation*}
    Now use (3.4.22) again to see that
    \begin{equation*} \cos^2(2x) = \frac{1 + \cos(4x)}{2} \end{equation*}
    which we insert:
    \begin{equation*} \frac{1 - \cos^2(2x)}{4} = \frac{1 - \frac{1 + \cos(4x)}{2}}{4} = \frac{\frac{2 - 1 - \cos(4x)}{2}}{4} = \boxed{\frac{1 - \cos(4x)}{8}} \end{equation*}
    Now there are no powers, and no trigonometric function is multipled by a trigonometric function.
    🔗
    🔗

    Subsection 3.4.4 Many Ways to Write the Same Answer

    Because of trigonometric identities, two expressions might look very different but actually be the same. It is sometimes not that easy to determine if two expressions are equivalent or not. One starting point is to compare the graphs on a graphing calculator, for example DESMOS.
    🔗

    Aside
    Study tip:
    🔗
    Suppose you have worked on a problem and you think you have a final answer, but your answer looks different from the answer given in some solution manual. You can graph both answers for example using DESMOS. If the graphs look different, then you know the answers are not the same. If the graphs look the same, then it is not a proof that the answers are equivalent, but it is a strong indication that they are.
    🔗
    You can not prove that two expressions are the same by looking at their graphs on a graphing calculator. But the graphs can be a gret first step to see if the expressions seem to be equal or not.
    🔗
